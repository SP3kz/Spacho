# Flowly – The Audacity AI Keyboard

Flowly is an AI-powered, neurodiverse-first keyboard system designed for clarity, automation, and expressive evolution.

## Features
- `/fy`, `/ly` rewriting syntax
- Talk-to-AI (Flowbot) integration
- Real-time SWOT self-optimization
- Script engine
- Cognitive NeuroMap™

## Setup
```bash
npm install
npm run start
```
